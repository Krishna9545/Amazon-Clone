📦 Project Title : 
Amazon Homepage Clone (HTML, CSS, JavaScript)


📝 Description : \
This project is a fully responsive Amazon homepage clone built using HTML, CSS, and JavaScript. 
It replicates the core UI and interactive features of the Amazon landing page, including a navigation bar,
search functionality, sidebar menu, image slider, product sections, and a detailed footer. 
The project focuses on front-end development, layout structuring, and DOM manipulation without using any frameworks.

The website includes a working search bar that redirects to Amazon search results, an auto-sliding hero banner,
interactive sidebar navigation, hover effects, and multiple product category sections similar to the original Amazon UI. 
This project is ideal for practicing responsive layouts, UI cloning, and JavaScript interactivity.


🚀 Features : 
Amazon-style responsive navbar
Functional search bar (redirects to Amazon results)
Auto image slider with navigation buttons
Sidebar menu with overlay
Product grid sections
Hover effects and UI interactions
Multi-column footer like Amazon
Clean UI using pure CSS
Vanilla JavaScript (no frameworks)


🛠️ Technologies Used : 
1) HTML5
2) CSS3
3) JavaScript (Vanilla JS)
4) Font Awesome Icons 


📁 Project Structure : 
├── index.html
├── style.css
├── script.js
└── images/


🎯 Learning Objectives : 
DOM manipulation using JavaScript
Responsive layout design
Sidebar & overlay implementation
Image slider functionality
UI cloning practice
Navigation & search interaction


▶️ How to Run : 
1) Download or clone the repository
2) Open index.html in your browser
3 ) Explore the UI and features
